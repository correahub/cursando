# **`counter()`** é uma função do CSS usada para **exibir o valor de um contador definido com `counter-reset` e `counter-increment`**, geralmente para listas numeradas personalizadas.

Exemplo:

```css
body {
  counter-reset: seção;
}

h2::before {
  counter-increment: seção;
  content: "Seção " counter(seção) ": ";
}
```

Aqui, cada `<h2>` exibirá **"Seção 1: ", "Seção 2: "**, etc., automaticamente.
