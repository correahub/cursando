**`hsl()`** é uma função do CSS usada para **definir cores usando matiz, saturação e luminosidade**.

Exemplo:

```css
div {
  background-color: hsl(0, 100%, 50%); /* vermelho puro */
}
```

- **H (Hue/matiz)**: posição da cor no círculo cromático (0 a 360°)
    
- **S (Saturation/saturação)**: intensidade da cor (0% a 100%)
    
- **L (Lightness/luminosidade)**: quão clara ou escura a cor é (0% a 100%)
