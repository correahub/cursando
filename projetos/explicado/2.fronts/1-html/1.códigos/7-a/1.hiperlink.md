# **`<a>` é um elemento usado para criar hiperlinks, que permite ao usuário navegar por entre as páginas de um site.** 

- `href` é o atributo define para onde o usuário será direcionado ao clicar no hiperlink.
    
- `target` é o atributo que define como o hiperlink ira funcionar ao ser clicado pelo usuário. Podendo abrir em uma nova página com o valor `_blank`. 
    
- `rel`: especifica a relação entre o documento atual e o hiperlink, útil principalmente para hiperlink externos, por exemplo `rel="noopener noreferrer"`.
