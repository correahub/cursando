# **`meter`** é um elemento usado para **representar visualmente um valor dentro de um intervalo conhecido** (como nível, capacidade ou desempenho).

Exemplo:

```html
<label for="combustivel">Combustível:</label>
<meter id="combustivel" value="70" min="0" max="100">70%</meter>
```

**Atributos principais:**

- **Value** é o atributo do meter que define o valor atual.
    
- **Min** é o atributo do meter que define o valor mínimo do intervalo.
    
- **Max** é o atributo do meter que define o valor máximo do intervalo.
    
- **Low** é o atributo do meter que define o valor considerado baixo.
    
- **High** é o atributo do meter que define o valor considerado alto.
    
- **Optimum** é o atributo do meter que define o valor ideal dentro do intervalo.
