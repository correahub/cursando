# **`input`** é um elemento usado para **criar campos de entrada de dados em formulários**.

Exemplo:

```html
<form>
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome" placeholder="Digite seu nome" required>
</form>
```

**Atributos principais:**

- **Type** é o atributo do input que define o tipo de campo (`text`, `password`, `email`, `number`, etc.).
    
- **Name** é o atributo do input que define o nome do campo para envio no formulário.
    
- **Value** é o atributo do input que define o valor inicial ou atual do campo.
    
- **Placeholder** é o atributo do input que define um texto de dica dentro do campo.
    
- **Required** é o atributo do input que define se o campo é obrigatório.
    
- **Disabled** é o atributo do input que define se o campo estará desativado.
    
- **Readonly** é o atributo do input que define se o campo será somente leitura.
    
- **Autocomplete** é o atributo do input que ativa ou desativa o preenchimento automático do navegador.
	
-  **Min** é o atributo do input que define o **valor mínimo permitido** para campos numéricos.
    
- **Max** é o atributo do input que define o **valor máximo permitido** para campos numéricos.
    
- **Minlength** é o atributo do input que define o **número mínimo de caracteres** permitido no campo de texto.
    
- **Maxlength** é o atributo do input que define o **número máximo de caracteres** permitido no campo de texto.
	
- **row** é o atributo que define que o numero de linhas.
	
- **columns** é o atributo define que o numero de colunas.
	
- **Low** é o atributo do meter que define o valor considerado baixo.
    
- **High** é o atributo do meter que define o valor considerado alto.
    
- **Optimum** é o atributo do meter que define o valor ideal dentro do intervalo.
